Vin
===

![Logo](resources/logo.svg)

[![Static Badge](https://img.shields.io/badge/Wikipedia-Decorator_pattern-blue?label=Wikipedia&link=https%3A%2F%2Fen.wikipedia.org%2Fwiki%2FVehicle_identification_number)](https://en.wikipedia.org/wiki/Vehicle_identification_number)

A type for representing a **vehicle identification number** (chassis number or frame number)

[![NuGet Version](https://img.shields.io/nuget/v/Skaar.Vin.svg)](https://www.nuget.org/packages/Skaar.Vin)

Use `Parse` or `TryParse` to create a `Vin` instance from a string.
Value serializes to and deserializes from JSON using `System.Text.Json`.
